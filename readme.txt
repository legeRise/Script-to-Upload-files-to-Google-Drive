It is a simple good to go script to upload files to google drive using python.
Remember! it is especially for " Google Service Account " which you can set up
using google cloud console.


Step 1: Once you have downloaded this repository and unzipped it in your local 
machine. open cmd(by typing cmd in the path bar) in the same folder and put this command
"pip install -r requirements.txt"

Step 2: Change the Script according to your needs

Requirements:

1. replace the uploader_service_account.json with your own json file path (for service
account)

2. change the file path and name for your file.

3. Change folder Id (gdrive folder where you want your file to be uploaded)


Step 3: In the command prompt enter " python uploaderScript.py". And done it will
start uploading
